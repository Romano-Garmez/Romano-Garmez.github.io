Box


This is a flattened image of a box I made. If you print it, cut it, and then subsequently fold it, all of the shapes should line up, creating a neat pattern. I did this out of abstract rounded shapes, because I like abstract art. It also felt like it would be much easier to make line up up I wasn't trying to fit a specific pattern. There wasn't a real artistic vision or anything, I just put in random shapes until I was happy.
